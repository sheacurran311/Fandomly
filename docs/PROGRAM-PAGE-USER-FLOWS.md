## State Transitions

### Program Status Flow

```
┌─────────┐
│  Draft  │ ◄── Initial state when created
└────┬────┘
     │
     │ [Publish]
     ▼
┌───────────┐
│ Published │ ◄── Accessible at /programs/:slug
└────┬──────┘
     │
     │ [Unpublish]
     ▼
┌─────────┐
│  Draft  │ ◄── Back to draft, not public
└────┬────┘
     │
     │ [Archive]
     ▼
┌──────────┐
│ Archived │ ◄── Hidden but preserved
└──────────┘
```

### Task Completion Flow

```
┌──────────────┐
│ Task Created │
└──────┬───────┘
       │
       │ [Fan clicks "Complete Task"]
       ▼
┌──────────────┐
│  Verification│
│   Required?  │
└──┬────────┬──┘
   │        │
   │ No     │ Yes
   │        │
   ▼        ▼
┌────────┐ ┌──────────────┐
│ Auto   │ │ Manual Review│
│Complete│ │  or Proof    │
└────┬───┘ └──────┬───────┘
     │            │
     │            │ [Approved]
     │            │
     └────────┬───┘
              ▼
     ┌─────────────────┐
     │ Task Completed  │
     │ Points Awarded  │
     │ Activity Logged │
     └─────────────────┘
```

## URL Structure

### Creator Routes (Authenticated)
```
/creator-dashboard/program-builder
  ├── (empty state - welcome screen)
  ├── ?program=:id (edit specific program)
  └── ?tab=overview|campaigns|tasks|customize|settings
```

### Public Routes (No Auth Required)
```
/programs/:slug
  ├── (default: dashboard tab)
  ├── ?tab=profile
  ├── ?tab=campaigns
  ├── ?tab=tasks
  └── ?tab=rewards
```

### API Routes
```
Creator (Authenticated):
  POST   /api/programs
  GET    /api/programs
  GET    /api/programs/:id
  PUT    /api/programs/:id
  DELETE /api/programs/:id
  POST   /api/programs/:id/publish
  POST   /api/programs/:id/unpublish
  POST   /api/programs/:programId/announcements
  GET    /api/programs/:programId/announcements
  PUT    /api/announcements/:id
  DELETE /api/announcements/:id

Public (No Auth):
  GET    /api/programs/public/:slug
  GET    /api/programs/:programId/activity
  GET    /api/programs/:programId/leaderboard
```
